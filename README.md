# Long-term_time_series_Fishery_census
　【和歌山県　長期時系列　漁業センサスです。】
◆漁業センサスとは
我が国の漁業の生産構造、就業構造を明らかにするとともに、漁村、水産物流通・加工業等の漁業を取り巻く実態と変化を総合的に把握するために、 統計法に基づき、５年ごとに実施している基幹統計調査です。
http://www.pref.wakayama.lg.jp/prefg/020300/jikeiretu/gyogyou/gyogyou.xls
